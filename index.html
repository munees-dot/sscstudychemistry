ഓർഗാനിക് സംയുക്തങ്ങളുടെ നാമകരണവും ഐസോമെറിസവും

രസതന്ത്രം · സ്റ്റാൻഡേർഡ് 10 · യൂണിറ്റ് 1 — സമ്പൂർണ പഠനോപകരണം

ചുരുക്ക സംഗ്രഹം

  • കാർബണിക സംയുക്തങ്ങളുടെ വൈവിധ്യത്തിന് കാരണം കാർബൺ ആറ്റങ്ങളുടെ പരസ്പര ബന്ധന ശേഷിയും (catenation), ഹൈഡ്രജൻ, ഓക്സിജൻ, നൈട്രജൻ തുടങ്ങിയ മൂലകങ്ങളുമായി ചേരാനുള്ള കഴിവുമാണ്.
  • ഒരേ തന്മാത്രാസൂത്രമുള്ള ഹൈഡ്രോകാർബണുകളുടെ ഘടനാവാക്യം (ശാഖകളുടെ ക്രമീകരണം) വ്യത്യസ്തമായിരിക്കാം.
  • IUPAC നാമകരണത്തിൽ ഏറ്റവും നീളം കൂടിയ കാർബൺ ചെയിൻ പ്രധാന ചെയിനായും ബാക്കിയുള്ളവ ശാഖ (ആൽക്കൈൽ ഗ്രൂപ്പ്) ആയും കണക്കാക്കുന്നു.
  • ശാഖകൾക്ക് ഏറ്റവും കുറഞ്ഞ സ്ഥാനസംഖ്യ ലഭിക്കുന്ന വിധത്തിലാണ് കാർബൺ ആറ്റങ്ങൾക്ക് നമ്പർ നൽകേണ്ടത്.
  • ഒരേ ശാഖ ഒന്നിലധികം തവണ വന്നാൽ ഡൈ, ട്രൈ, ടെട്രാ എന്നീ പ്രത്യയങ്ങൾ ഉപയോഗിക്കണം; സ്ഥാനസംഖ്യകൾ കോമ ഉപയോഗിച്ച് വേർതിരിക്കണം.
  • അക്കങ്ങളും അക്ഷരങ്ങളും തമ്മിൽ ഹൈഫൻ (–) ഉപയോഗിച്ച് വേർതിരിക്കണം.
  • ദ്വിബന്ധനം (C=C) ഉള്ളവ ആൽക്കീനുകൾ (–ene), ത്രിബന്ധനം (C≡C) ഉള്ളവ ആൽക്കൈനുകൾ (–yne) എന്നറിയപ്പെടുന്നു. ബന്ധനത്തിന് ഏറ്റവും കുറഞ്ഞ സ്ഥാനസംഖ്യ ലഭിക്കണം.
  • ഫങ്ഷണൽ ഗ്രൂപ്പ് എന്നത് ഒരു സംയുക്തത്തിന്റെ സവിശേഷ രാസ-ഭൗതിക ഗുണങ്ങൾ നിർണ്ണയിക്കുന്ന ആറ്റം/ആറ്റഗ്രൂപ്പ് ആണ്.
  • ഫങ്ഷണൽ ഗ്രൂപ്പ് ഉള്ള ചെയിനിനെ പ്രധാന ചെയിനായി കണക്കാക്കണം; ഫങ്ഷണൽ ഗ്രൂപ്പിന് ഏറ്റവും കുറഞ്ഞ സ്ഥാനസംഖ്യ നൽകണം.
  • ഒരേ തന്മാത്രാസൂത്രവും വ്യത്യസ്ത ഗുണങ്ങളുമുള്ള സംയുക്തങ്ങളെ ഐസോമെറുകൾ എന്നു വിളിക്കുന്നു; ഈ പ്രതിഭാസത്തെ ഐസോമെറിസം എന്നു പറയുന്നു.

രണ്ട് ഉദാഹരണ ഹൈഡ്രോകാർബണുകൾ

CH₃–CH₂–CH₂–CH₂–CH₃  (ഹൈഡ്രോകാർബൺ I — നീളൻ ചെയിൻ)

CH₃–CH(CH₃)–CH₂–CH₃  (ഹൈഡ്രോകാർബൺ II — ശാഖയുള്ളത്)

ഇവ രണ്ടും C₅H₁₂ എന്ന ഒരേ തന്മാത്രാസൂത്രമുള്ളവയാണ്, പക്ഷേ ഘടനാവാക്യം വ്യത്യസ്തമാണ് — ഇത് ചെയിൻ ഐസോമെറിസത്തിന്റെ അടിസ്ഥാനമാണ്.

⚡ ക്വിക്ക് റിവിഷൻ ഷീറ്റ് — ഒറ്റ നോട്ടത്തിൽ മുഴുവൻ യൂണിറ്റ്

ബെൻസീൻ C₆H₆  ·  ഫീനോൾ C₆H₅OH  ·  ബെൻസോയിക് ആസിഡ് C₆H₅COOH

📌 പരീക്ഷാ ടിപ്പ്: ഏത് സംയുക്തം കണ്ടാലും ആദ്യം ഫങ്ഷണൽ ഗ്രൂപ്പ് കണ്ടെത്തുക → നീളൻ ചെയിൻ (ഫങ്ഷണൽ ഗ്രൂപ്പ് ഉൾപ്പെടെ) കണ്ടെത്തുക → ഫങ്ഷണൽ ഗ്രൂപ്പിന്/ബന്ധനത്തിന് കുറഞ്ഞ നമ്പർ വരും വിധം എണ്ണുക → ശാഖകൾ ചേർത്ത് പേര് പൂർത്തിയാക്കുക.

നാമകരണ നിയമങ്ങളും സൂത്രവാക്യങ്ങളും

1. ശാഖയുള്ള ആൽക്കെയ്‌നുകൾ

ശാഖയുടെ സ്ഥാനസംഖ്യ + ഹൈഫൻ + ആൽക്കൈൽ ഗ്രൂപ്പിന്റെ പേര് + പദമൂലം + പിൻപ്രത്യയം (ഐൻ)
  • ഘട്ടം 1: ഏറ്റവും കൂടുതൽ കാർബൺ ആറ്റങ്ങൾ അടങ്ങിയ നീളം കൂടിയ ചെയിൻ പ്രധാന ചെയിനായി തിരഞ്ഞെടുക്കുക.
  • ഘട്ടം 2: ശാഖയുള്ള കാർബൺ ആറ്റത്തിന് ഏറ്റവും കുറഞ്ഞ സ്ഥാനസംഖ്യ വരുന്ന രീതിയിൽ (ഇടത്ത്/വലത്ത്) നമ്പർ ചെയ്യുക.
  • ഘട്ടം 3: ഒരേ ശാഖ ഒന്നിലധികം വന്നാൽ ഡൈ (2), ട്രൈ (3), ടെട്രാ (4) ചേർത്ത് സ്ഥാനസംഖ്യകൾ കോമയിട്ട് വേർതിരിക്കുക.
  • ഘട്ടം 4: അക്കവും അക്ഷരവും തമ്മിൽ ഹൈഫൻ ഉപയോഗിക്കുക.

ആൽക്കൈൽ ഗ്രൂപ്പുകൾ (പട്ടിക 1.1)

ആൽക്കൈൽ ഗ്രൂപ്പ്ഘടനാവാക്യം
മീഥൈൽ (Methyl)–CH₃
ഈഥൈൽ (Ethyl)–CH₂–CH₃
പ്രൊപ്പൈൽ (Propyl)–CH₂–CH₂–CH₃
ആൽക്കൈൽ ഗ്രൂപ്പിന്റെ പേര് = ശാഖയിലെ കാർബൺ ആറ്റത്തിന്റെ എണ്ണത്തിനുള്ള പദമൂലം + ഐൽ (yl)

2. അപൂരിത ഹൈഡ്രോകാർബണുകൾ

ആൽക്കീൻ = പദമൂലം + ഹൈഫൻ + ദ്വിബന്ധനത്തിന്റെ സ്ഥാനം + ഹൈഫൻ + ഈൻ (ene)
ആൽക്കൈൻ = പദമൂലം + ഹൈഫൻ + ത്രിബന്ധനത്തിന്റെ സ്ഥാനം + ഹൈഫൻ + ഐൻ (yne)

ബന്ധനത്തിന് ഏറ്റവും കുറഞ്ഞ സ്ഥാനസംഖ്യ ലഭിക്കുന്ന വിധം നമ്പർ ചെയ്യണം.

  • ഉദാ: CH₂=CH–CH₂–CH₃ → ബ്യൂട്ട്–1–ഈൻ (But-1-ene)
  • ഉദാ: CH₃–CH=CH–CH₃ → ബ്യൂട്ട്–2–ഈൻ (But-2-ene)
  • ഉദാ: CH≡C–CH₂–CH₃ → ബ്യൂട്ട്–1–ഐൻ (But-1-yne)

3. ഫങ്ഷണൽ ഗ്രൂപ്പ് പ്രത്യയങ്ങൾ (പദമൂലത്തിലെ 'e' മാറ്റി ചേർക്കുന്നത്)

ഫങ്ഷണൽ ഗ്രൂപ്പ്പൊതു നാമംരൂപീകരണ സൂത്രം
–OHആൽക്കഹോൾAlkane – e + ol → Alkanol
–COOHകാർബോക്സിലിക് ആസിഡ്Alkane – e + oic acid → Alkanoic acid
–CHOആൽഡിഹൈഡ്Alkane – e + al → Alkanal
>C=Oകീറ്റോൺAlkane – e + one → Alkanone

4. ഹാലോ സംയുക്തങ്ങൾ & ഈഥറുകൾ

ഹാലോ ഗ്രൂപ്പിന്റെ സ്ഥാനം + ഹൈഫൻ + ഹാലോ ഗ്രൂപ്പിന്റെ പേര് + ആൽക്കെയ്‌നിന്റെ പേര്

ഉദാ: CH₃–CH₂–CH₂–Cl → 1–ക്ലോറോപ്രൊപ്പെയ്ൻ (1-Chloropropane)

ആൽക്കോക്സി ഗ്രൂപ്പ് (–O–R) + ആൽക്കെയ്‌നിന്റെ പേര് → ഈഥർ നാമം

ഉദാ: CH₃–O–CH₂–CH₃ → മീഥോക്സിഈഥെയ്ൻ (Methoxyethane)

പൊതു ഫോർമുല (എല്ലാ ഭാഗങ്ങളും ചേർത്ത്)

ശാഖയുടെ സ്ഥാനസംഖ്യ + ഹൈഫൻ + ആൽക്കൈൽ ഗ്രൂപ്പിന്റെ പേര് + പദമൂലം + ഹൈഫൻ + ഫങ്ഷണൽ ഗ്രൂപ്പിന്റെ സ്ഥാനസംഖ്യ + ഹൈഫൻ + പിൻപ്രത്യയം

ഫങ്ഷണൽ ഗ്രൂപ്പുകൾ — സമ്പൂർണ പട്ടിക

ഫങ്ഷണൽ ഗ്രൂപ്പ്ഗ്രൂപ്പിന്റെ പേര്പൊതു നാമംപിൻ/മുൻ പ്രത്യയംഉദാഹരണം
–OHഹൈഡ്രോക്സിൽആൽക്കഹോൾ–olCH₃OH മെഥനോൾ
–COOHകാർബോക്സിൽകാർബോക്സിലിക് ആസിഡ്–oic acidCH₃COOH എഥനോയിക് ആസിഡ്
–CHOആൽഡിഹൈഡ്ആൽഡിഹൈഡ്–alCH₃CHO എഥനാൽ
>C=Oകീറ്റോകീറ്റോൺ–oneCH₃COCH₃ പ്രൊപ്പനോൺ
–O–Rആൽക്കോക്സിഈഥർalkoxy–CH₃OCH₃ മീഥോക്സിമീഥെയ്ൻ
–F, –Cl, –Br, –Iഹാലോഹാലോ സംയുക്തങ്ങൾhalo–CH₃CH₂Cl ക്ലോറോഈഥെയ്ൻ

അരോമാറ്റിക് സംയുക്തങ്ങൾ

സംയുക്തംഘടനഫോർമുല
ബെൻസീൻആറംഗ വലയ ഘടന, ഒന്നിടവിട്ട ദ്വിബന്ധനങ്ങൾC₆H₆
ഫീനോൾബെൻസീൻ വലയത്തിൽ –OH ഗ്രൂപ്പ്C₆H₅–OH
ബെൻസോയിക് ആസിഡ്ബെൻസീൻ വലയത്തിൽ –COOH ഗ്രൂപ്പ്C₆H₅–COOH

ഓർക്കുക ഫങ്ഷണൽ ഗ്രൂപ്പ് ഉള്ള കാർബൺ ആറ്റത്തിന് ചെറിയ സ്ഥാനസംഖ്യ ലഭിക്കുന്ന വിധം ചെയിൻ നമ്പർ ചെയ്യണം.

ഐസോമെറിസം — തരങ്ങൾ

നിർവചനം: ഒരേ തന്മാത്രാസൂത്രവും വ്യത്യസ്ത രാസ-ഭൗതിക സ്വഭാവവുമുള്ള സംയുക്തങ്ങളെ ഐസോമെറുകൾ എന്നും ഈ പ്രതിഭാസത്തെ ഐസോമെറിസം എന്നും വിളിക്കുന്നു. ഇവയുടെ ഘടനാവാക്യങ്ങൾ വ്യത്യസ്തമായിരിക്കും.

1. ചെയിൻ ഐസോമെറിസം (Chain isomerism)

ഒരേ തന്മാത്രാസൂത്രമുള്ളവയും കാർബൺ ചെയിനിന്റെ ഘടനയിൽ വ്യത്യസ്തമായി പുലർത്തുകയും ചെയ്യുന്ന സംയുക്തങ്ങൾ.

CH₃CH₂CH₂CH₃ (n-ബ്യൂട്ടെയ്ൻ)
CH₃CH(CH₃)CH₃ (2-മീഥൈൽപ്രൊപ്പെയ്ൻ)

2. പൊസിഷൻ ഐസോമെറിസം (Position isomerism)

ഒരേ തന്മാത്രാസൂത്രവും ഒരേ ഫങ്ഷണൽ ഗ്രൂപ്പും ഉള്ളവയിൽ ഫങ്ഷണൽ ഗ്രൂപ്പ് / ബന്ധനത്തിന്റെ സ്ഥാനം വ്യത്യസ്തമെങ്കിൽ.

CH₃CH₂CH₂OH (പ്രൊപ്പൻ-1-ഓൾ)
CH₃CH(OH)CH₃ (പ്രൊപ്പൻ-2-ഓൾ)

3. ഫങ്ഷണൽ ഐസോമെറിസം (Functional isomerism)

ഒരേ തന്മാത്രാസൂത്രമുള്ളതും എന്നാൽ ഫങ്ഷണൽ ഗ്രൂപ്പ് വ്യത്യസ്തമായതുമായ സംയുക്തങ്ങൾ.

CH₃CH₂OH (എഥനോൾ — ആൽക്കഹോൾ)
CH₃–O–CH₃ (മീഥോക്സിമീഥെയ്ൻ — ഈഥർ)

4. മെറ്റാമെറിസം (Metamerism)

ഒരേ തന്മാത്രാസൂത്രവും ഒരേ ഫങ്ഷണൽ ഗ്രൂപ്പും (–O– അല്ലെങ്കിൽ >C=O) ഉള്ളതും എന്നാൽ ആ ഗ്രൂപ്പിന്റെ ഇരുവശവുമുള്ള ആൽക്കൈൽ ഗ്രൂപ്പുകൾ വ്യത്യസ്തമായതുമായ ഐസോമെറിസം.

CH₃CH₂–O–CH₂CH₃ (ഈഥോക്സിഈഥെയ്ൻ)
CH₃–O–CH₂CH₂CH₃ (മീഥോക്സിപ്രൊപ്പെയ്ൻ)

ഓർമ്മ സഹായി

തരംഎന്ത് വ്യത്യാസപ്പെടുന്നു
ചെയിൻകാർബൺ ചെയിനിന്റെ ഘടന (ശാഖ)
പൊസിഷൻഫങ്ഷണൽ ഗ്രൂപ്പ്/ബന്ധനത്തിന്റെ സ്ഥാനം
ഫങ്ഷണൽഫങ്ഷണൽ ഗ്രൂപ്പ് തന്നെ വ്യത്യസ്തം
മെറ്റാമെറിസം–O–/>C=O ഗ്രൂപ്പിന്റെ ഇരുവശത്തെയും ആൽക്കൈൽ ഗ്രൂപ്പുകൾ

ക്വിക്ക് ക്വിസ്

സ്കോർ: 0 / 0  ·  ചോദ്യം 1 / 0

രസതന്ത്രം — മാതൃകാ ചോദ്യപേപ്പർ

യൂണിറ്റ് 1: ഓർഗാനിക് സംയുക്തങ്ങളുടെ നാമകരണവും ഐസോമെറിസവും

സമയം: 45 മിനിറ്റ് ആകെ സ്കോർ: 20
വിഭാഗം A — 1 സ്കോർ വീതം (5 × 1 = 5)
1. –COOH ഫങ്ഷണൽ ഗ്രൂപ്പുള്ള സംയുക്തങ്ങളെ എന്ത് പേരിൽ അറിയപ്പെടുന്നു?
2. CH₃–CH₂–CH₂–CH₃ എന്ന സംയുക്തത്തിന്റെ IUPAC നാമം എഴുതുക.
3. ഈഥറുകളിലെ ഫങ്ഷണൽ ഗ്രൂപ്പ് ഏത്?
4. ദ്വിബന്ധനമുള്ള ഹൈഡ്രോകാർബണുകളുടെ പിൻപ്രത്യയം എന്ത്?
5. ഐസോമെറിസം എന്നാൽ എന്ത്?
വിഭാഗം B — 2 സ്കോർ വീതം (4 × 2 = 8)
6. CH₃–CH(CH₃)–CH₂–CH₃ എന്ന സംയുക്തത്തിന്റെ IUPAC നാമം കണ്ടെത്തുക.
7. 2,3–ഡൈമീഥൈൽപെന്റെയ്ൻ എന്ന സംയുക്തത്തിന്റെ ഘടനാവാക്യം എഴുതുക.
8. ചെയിൻ ഐസോമെറിസവും പൊസിഷൻ ഐസോമെറിസവും തമ്മിലുള്ള വ്യത്യാസം എന്ത്?
9. CH₃CH₂OH ഉം CH₃OCH₃ ഉം ഏത് തരം ഐസോമെറിസം പ്രദർശിപ്പിക്കുന്നു? കാരണം എഴുതുക.
വിഭാഗം C — 3 സ്കോർ വീതം (2 × 3 = 6)
10. താഴെ നൽകിയിരിക്കുന്ന ഹൈഡ്രോകാർബണിന്റെ IUPAC നാമം ഘട്ടം ഘട്ടമായി കണ്ടെത്തുക:
CH₃–CH₂–CH(CH₃)–CH₂–CH₃
11. ഫങ്ഷണൽ ഗ്രൂപ്പ് എന്നാൽ എന്ത്? ഏതെങ്കിലും മൂന്ന് ഫങ്ഷണൽ ഗ്രൂപ്പുകളും അവയുടെ പൊതു നാമവും എഴുതുക.
വിഭാഗം D — ഉപന്യാസ തരം (1 × 1 = 1)
12. പെന്റെയ്നിന്റെ (C₅H₁₂) മൂന്ന് ചെയിൻ ഐസോമെറുകളുടെയും ഘടനാവാക്യങ്ങളും IUPAC നാമങ്ങളും എഴുതുക.
